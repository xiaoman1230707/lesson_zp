提示
给你一个字符串 s，找到 s 中最长的 回文 子串。

 

示例 1：

输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
示例 2：

输入：s = "cbbd"
输出："bb"


###
中心扩散法：
回文字符串意味着，把左右两边都减掉一个也为回文字符串。他有一个虚拟的中心线，奇数个则中间为一个，偶数个则中间为两个字符，由中心向左右一个一个扩散，就能找出以这个为中心的最长回文串。
那么我们遍历字符串，每一个i 都查找他的中心 最长回文串，比较就能得到整个字符中最长的回文串。
注意：因为有奇偶的不同，我们 传递中心 求最长回文串的函数可以设置为共有的，传递两个指针来作为中心，一样为奇，差一为偶，main中两个都求取大的就可以
