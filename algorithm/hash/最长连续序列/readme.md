128. 最长连续序列
已解答
中等

给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。

请你设计并实现时间复杂度为 O(n) 的算法解决此问题。

 

示例 1：

输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。
示例 2：

输入：nums = [0,3,7,2,5,8,4,6,0,1]
输出：9
示例 3：

输入：nums = [1,0,1,2]
输出：3

###
整个nums中如果排好序，那就会有很多个连续数字的序列
要找到这些数字连续的序列，可以找到每一段的第一个数字n，就能定位他的位置和长度。
即在nums中没有 n - 1这个数字，可以找到第一个；
向后一个一个n + 1寻找，即可得到这段连续数字的长度；
记录比较这些长度得到最大

使用Set做hash的存储，可以做到O(1)的查询，且不会二次存入同样的数据，
注意，遍历时遍历set而不是nums，因为set去过重，长度更短。
