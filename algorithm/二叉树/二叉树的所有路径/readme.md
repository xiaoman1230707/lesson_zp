257. 二叉树的所有路径

给你一个二叉树的根节点 root ，按 任意顺序 ，返回所有从根节点到叶子节点的路径。

叶子节点 是指没有子节点的节点。

 
示例 1：


输入：root = [1,2,3,null,5]
输出：["1->2->5","1->3"]
示例 2：

输入：root = [1]
输出：["1"]


###
递归遍历每一个节点，参数传递节点和一个路径字符串，每遍历到一个节点就在路径串上加上该节点的值，为空则不加，并判断他的两个子节点是否为空，都为空那么就代表此节点为叶子结点，将传递到此的路径字符串push进全局的res。最后返回false表示此节点不为空。
