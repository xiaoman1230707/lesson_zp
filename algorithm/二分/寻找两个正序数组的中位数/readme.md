4. 寻找两个正序数组的中位数
已解答
困难

给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。

算法的时间复杂度应该为 O(log (m+n)) 。

 
示例 1：

输入：nums1 = [1,3], nums2 = [2]
输出：2.00000
解释：合并数组 = [1,2,3] ，中位数 2
示例 2：

输入：nums1 = [1,2], nums2 = [3,4]
输出：2.50000
解释：合并数组 = [1,2,3,4] ，中位数 (2 + 3) / 2 = 2.5


###
二分法：
核心思想为：求两个数组的第k小个数字时，不需要一个一个比较排除，而是一半一半排除，比较在两个数组k/2的值，更小的那个前面的也一定 比另一个小，这样就是一次性排除多个不符合的目标。

可以使用递归的方法操作，参数值要有记录两数组排除后的开始节点 和 需要求得第k小个数。
递归结束标志为k为1，那么返回两开始节点更小者。
或者一边数组为空，那么第k小的数为另一个数组开始节点向后数k - 1个。
比较两数组开始节点向后数k / 2 - 1 的大小，不够则为最后一个，
小的那边，减少k 的大小为排除掉的更小个数。修改开始位置为比较位置的下一个。继续递归下去。
