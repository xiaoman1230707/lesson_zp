347. 前 K 个高频元素
已解答
中等

给你一个整数数组 nums 和一个整数 k ，请你返回其中出现频率前 k 高的元素。你可以按 任意顺序 返回答案。

 

示例 1：

输入：nums = [1,1,1,2,2,3], k = 2

输出：[1,2]

示例 2：

输入：nums = [1], k = 1

输出：[1]

示例 3：

输入：nums = [1,2,1,2,1,2,3,1,3,2], k = 2

输出：[1,2]

 

提示：

1 <= nums.length <= 105
-104 <= nums[i] <= 104
k 的取值范围是 [1, 数组中不相同的元素的个数]
题目数据保证答案唯一，换句话说，数组中前 k 个高频元素的集合是唯一的
 

进阶：你所设计算法的时间复杂度 必须 优于 O(n log n) ，其中 n 是数组大小。



####
首先做到：
统计元素出现频率
对频率排序
找出前K个高频元素

出现频率可以用map来统计；
而一个长度为k优先级队列即可满足后两个；
维护一个小顶堆，固定他的长度为k，将map中的元素频率一一push进去，当超出需要的k个时，就会pop出最小的频率，直到其中的k个，即为频率前k高的元素。

小顶堆的实现：由一个完全二叉树组成，那么即可用一个一维数组存储数据。
堆元素的两个操作：
上滤： 锁定一个子节点与父节点做比较，较小者在上，直到大于父节点或者为最顶端。
下滤： 锁定顶端节点 与较小子节点比较，更大者在下，直到小于所有子节点或者为叶子结点。

小顶堆的两个主要方法，pop始终将堆内最小的值弹出，push新的值进去，那么就维护了一个队头一直是最小值得队列。

pop：将最顶端元素弹出，并将实现数组的最后一个换到顶端，对这个顶端元素进行下滤。
push：将新元素加入到实现数组最后，并对该元素进行上滤。
