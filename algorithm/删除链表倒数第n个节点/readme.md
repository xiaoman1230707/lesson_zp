19. 删除链表的倒数第 N 个结点
已解答
中等

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

 


输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]
示例 2：

输入：head = [1], n = 1
输出：[]
示例 3：

输入：head = [1,2], n = 1
输出：[1]

####
如果要删除倒数第n个节点，让fast移动n步，然后让fast和slow同时移动，直到fast指向链表末尾。删掉slow所指向的节点就可以了。
删除节点需要找到上一个节点，所以fast移动n步。
使用虚拟头结点，这样方便处理删除实际头结点的逻辑
