提示
给定一个字符串 s ，请你找出其中不含有重复字符的 最长 子串 的长度。

 

示例 1:

输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。
示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。


###
记录下最长无相同字符串长度，两个指针圈定目标字符串，后指针向后移动，判断新加入字符是否被前字符包括，是则移动前指针，直到不含有，加入新字符。
注意，当后指针指到最后一个时，那么记录的最长值不可能再变大了，所以while 的结束标志为右指针超出数组长度；
