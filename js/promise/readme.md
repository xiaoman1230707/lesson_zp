# Promise JS 异步

- 线程
  - 执行代码的最小单位
  - 进程是分配资源的最小单位
  - 进程启动线程，把代码执行任务交给他
  - JS是单线程
    - 同步代码
      从上到下
      console let for ...
      ms级别
    - 异步代码
      时间开销
      可能和阅读顺序不一样
    - 单线程
      JS语言简单好学
      JS以负责事件，页面更新...
      一旦碰上耗时性任务，先放入event loop，接着往后执行。

- Promise
  es6 提供的异步变同步的高级工具类
  提供了一个 then 方法控制流程 等下
  new Promise 接受一个函数作为参数
    用于立即执行异步任务，执行完后，调用resolve() 成功
    .then里的函数会立即执行
    reject() 失败，会触发catch里的函数


pending（等待中）：初始状态，既没有被兑现，也没有被拒绝。
fulfilled（已成功）：操作成功完成。
rejected（已失败）：操作失败。
