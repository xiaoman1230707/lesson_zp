## 解构

- 一次性声明多个变量
- 解构了数组，甚至还可以解构嵌套数组。
- 解构对象 只要左右两边一致

## reset 运算符 ...
- 可以将数组或类数组对象的剩余元素赋值给一个变量
- 只能用在数组或类数组对象的解构赋值中

## 对象的属性简写
- 当对象的属性名和变量名相同时，可以省略属性名
- 当对象的属性值是变量时，可以省略属性值
