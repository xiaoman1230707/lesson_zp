# 对象字面量

## JS是最有表现力的脚本语言
- 不需要想java/c++ 一样，先定义类，早期的js class关键字都没有
  {} 是对象字面量
  [] 是数组
- js提供了对象(object)字面量(字面意义上就知道这是个对象)

## 面向对象
- 对象由属性和方法构成
- 简单的面向对象(只送花)
- 复杂的人际关系的面向对象

## JS数据类型
- 字符串 String
- 数值 Number
- 布尔值 Boolean
- 空值 Null
- 未定义 Undefined
- 对象 Object

## 设计模式 Proxy 代理
- 面向**接口(InterFace)**的编程，代码就灵活且强大
- zzp -> xm 送花 大概率会寄
- 添加一个xh对象字面量,xh拥有和xm一样的reciveFlower方法
- zzp -> xm ,zzp -> xh 送花
  xh 可以代理 xm 收花
  xm xh 实现了一样的reciverFlower接口
- 这就是代理模式
