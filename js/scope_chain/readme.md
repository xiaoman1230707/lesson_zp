## 词法作用域 以及 词法作用域链
指作用域是由代码中函数声明的位置来决定的，词法作用域是静态的作用域
作用域链也叫词法作用域链，静态的，只和函数声明的位置相关，在编译阶段就决定好的，与调用无关

## 闭包 closure 
无处不在的高级概念 
静态的，词法作用域 
- 函数在运行时 也有一种情况
  就好像函数运行时不是在他的声明的位置时，就会向外顺着作用域链查找

- 闭包基于词法作用域的理解
- 形成条件 函数嵌套函数
- 被闭包的函数要在外部可以访问到 return出来
- 被闭包的函数执行的时候能找到定义它的时候的执行上下文中的变量，即使在定义它的执行上下文已经出栈了

foo函数执行完后 期执行上下文从栈顶弹出了，但是由于返回的setname getname 使用了foo函数内部的变量myname 和 test1 ，这两个变量依然在内存中,有点像给getname setname方法背的一个专属背包
这个背包为闭包，闭包里面的变量叫自由变量

闭包closure 直到程序结束才会被销毁
