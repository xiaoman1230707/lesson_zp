## 防抖和节流区别
在一定时间内 只执行最后一次 防抖(setTimeout)
每隔一定时间执行一次 节流(setInterval)

函数节流就像是fps游戏里的射速，就算一直按着鼠标射击，也只会在规定射速内射出子弹。

## 总结
- 函数的防抖和节流都是防止某一时间频繁触发，但是原理不一样
- debounce 防抖 某短时间内只执行一次
  - 搜索、用户不断输入值时，用防抖来节约请求资源。
- throttle 节流 间隔时间执行
  - 滚动加载，用户不断滚动页面时，用节流来节约请求资源。
