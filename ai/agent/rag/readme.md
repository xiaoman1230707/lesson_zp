## 为什么是向量表达？

- 关键词的文本匹配不能实现语义搜索
  - 查询文章提到的水果，那么关键词就只能匹配‘水果’，而不能找到‘香蕉’‘苹果’等
- 向量 Vector [0,1,2,...]
  - 用数字表达存储的信息 一个维度(如水果，数码等)的分值表示他在这个维度上的相关性。当维度足够多时，就能表达一个事物所有的信息。
    食用性 0 无 ~ 1 高
    硬度 0 液体 ~ 1 固体
    水果 [0.9,0.3] 食用性高 硬度偏低
    苹果 [0.8,0.4] 食用性高 硬度偏高
    香蕉 [0.9,0.1]
    石头 [0.1,0.9]
- 语义搜索的流程
  - 向量每个维度都有独特的语义 (食用性，硬度,...)
  - 可视化 空间
  - cosin 计算
