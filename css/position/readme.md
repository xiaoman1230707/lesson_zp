# 定位
- 离开文档流
  - display:none
  - position:absolute
  文档流是html元素默认的布局方式，块级元素垂直排列，行内元素水平排列，遵循从上到下，从左到右的自然顺序。
  - 元素在文档流中，离开？
- position:relative
  - 相对定位元素，是相对于元素在文档流中的原来位置进行定位
  - 不会离开文档流，原来的标准流的位置继续占有，后面的盒子仍然以标准的流的方式对待他
- display:none
  - 隐藏元素，不占用空间,不会影响到其他元素的的布局
- position:absolute
  - 绝对定位
  - 需要一个参考标准，参考标准是最近的有定位的父元素如果父容器没有 则参考body. 离他最近的position属性不是static的元素
  - 会离开文档流
- position:fixed
  - 固定定位
  - 以浏览器为准定位，固定也会脱离文档流
  - 固定定位的盒子不会随着浏览器的滚动而滚动

- position:sticky
  - 粘性定位，是相对定位和固定定位的结合体
  - 根据滚动的位置在relative和fixed之间切换
  - 他的行为像relative定位，知道滚动到指定的阈值，他就会像fixed定位一样固定在该位置。
  - 粘性定位的元素在滚动到指定位置时，会固定在指定位置，不会离开文档流

- position:static
  - 静态定位
  - 元素的定位是根据标准流的位置进行定位的
  - 静态定位的元素不会脱离文档流
  - 用它取消元素的定位属性
